komodo-rubyonrails
==================

komodo editor config for rubyonrails

![Screenshot](https://github.com/brookzhang/komodo-rubyonrails/raw/master/screen.jpg)

Summary
-------
Komodo is a free editor , running on Windows & Linux & Mac
[Download](http://www.activestate.com/komodo-edit/downloads )

The scheme in screeshot is my config ,please copy RubyOnRails.ksf to [KomodoPath\lib\support\schemes],then follow these steps: Edit->Preferences->Fonts and Colors , select RubyOnRails from the dropdownlist, nearby [Scheme:(system schemes are in bold)]



中文版说明
----------

Komodo是一款Free的Editor,此公司主要靠Komodo IDE收费赚钱, Komodo editor纯免费,但是已经非常好用了 
Komodo支持windows,linux,mac ,[下载地址](http://www.activestate.com/komodo-edit/downloads ) 

我配置了在railscast里面看到的textmate风格,感觉很舒服. 

此风格设定,包括对ruby的自动完成等功能,需要自行设定,我的设定在附件,请下载后把里面的RubyOnRails.ksf放到你的Komodo安装目录下的\lib\support\schemes中. 然后进入Komodo 按如下设定:Edit->Preferences->Fonts and Colors 右侧中间的Scheme:(system schemes are in bold) 的下拉列表中选择RubyOnRails 

需说明:复制到\lib\support\schemes里的ksf文件即system schemes,不能修改. 如果你做了调整,就会提示你另存新的scheme,自己新建的scheme保存在C:\Users\youraccount\AppData\Local\ActiveState\KomodoEdit\7.1\schemes里. 

虽然没有IDE复杂,但是简单实用,且windows linux通用,大家可以试试看.
